# Handwritten Digit Recognition

This project is a **Handwritten Digit Recognition System** built using **Python, TensorFlow, and Tkinter**.
It allows users to draw a digit on a canvas and predicts the number using a trained **MNIST deep learning model**.

## How it Works

* Draw a digit using the mouse
* Click **Recognize**
* The model predicts the digit
* Click **Clear** to draw again

---

## Description

The application uses a **Convolutional Neural Network (CNN)** trained on the **MNIST dataset** to recognize handwritten digits in real time.
